# Smart Parking System

A parking management system that allows users to book parking slots, track entry and exit, and calculate charges based on time.

## Features
- User login and signup
- Slot booking system
- Entry and exit tracking
- Time-based billing
- Slot availability display
- Basic dashboard for usage and revenue

## Tech Stack
- HTML, CSS, JavaScript (Frontend)
- Backend (to be implemented)
- Database (planned)
